DISEÑO WEB

¿Qué es el diseño web?

El diseño web es una actividad que consiste en la planificación, diseño, implementación y mantenimiento de sitios web. No es simplemente la implementación del diseño convencional ya que se abarcan diferentes aspectos como el diseño gráfico web, diseño de interfaz y experiencia de usuario, como la navegabilidad, interactividad, usabilidad, arquitectura de la información; interacción de medios, entre los que podemos mencionar audio, texto, imagen, enlaces, video y la optimización de motores de búsqueda.

Importancia de tener un diseño web

Todo tiene una función, un orden, todos los elementos que conforman una web siguen una estrategia pensada para el usuario final. Hay que hacer que el usuario al navegar en una web se comporte como queremos que lo haga en función de sus necesidades. Es decir, pensar en un diseño web en función del usuario y obtener de él la reacción que nos interesa. El diseñador se encargará de que el usuario al entrar a tu web navegue facilmente por ella, ya sea un blog o una pagina de ventas y que permanezca el mayor tiempo posible (que es, lo que busca tu negoció) y se sienta comodo estando en tu página.

La unión de un buen diseño con una jerarquía bien elaborada de contenidos, aumenta la eficiencia de la web como canal de comunicación e intercambio de datos, que brinda posibilidades como el contacto directo entre el productor y el consumidor de contenidos.

Tener un diseño de tu página web, te ahorra futuros gastos extras por cambios de último momento, es mejor pagar por cambios en diseño a pagar por cambios en codigo del desarrollador.

El diseño web adaptable (en inglés, responsive) es una filosofía de diseño web que tiene como objetivo adaptar una misma página web a distintos tamaños de pantalla, dispositivos y orientaciones. Así puedes tener un panorama completo de como verán tus visitantes la página web en cualquier dispositivo.
